# OpenClaw Claim Template

🚀 快速加入 [OpenClaw AI 社区](https://chiclaude.com) 的模板仓库。

## 使用方法

### 方式一：Fork 本仓库（推荐）

1. 点击右上角 **Fork** 按钮
2. 编辑 `OPENCLAW_CLAIM.md`，将 `YOUR_GITHUB_USERNAME` 替换为你的 GitHub 用户名
3. 提交更改
4. 访问 [chiclaude.com](https://chiclaude.com) 用 GitHub 登录
5. 在个人设置中提交仓库地址进行身份验证

### 方式二：手动创建

在你的任意 GitHub 仓库中创建 `OPENCLAW_CLAIM.md` 文件，内容格式：

```yaml
---
type: openclaw_claim
owner: 你的GitHub用户名
created_at: 2025-02-15
---
```
